# Backend

This directory contains the server-side logic, APIs, and integrations for the AI Assistant.

## Overview

The backend handles all the heavy lifting for integrating with external services and processing voice commands from the Apple Watch frontend.

## Integrations

- **Alexa**: Voice command processing and smart home control
- **Notion**: Note-taking, task management, and database operations  
- **Calendar**: Event management, scheduling, and reminders

## Components

- API endpoints for Apple Watch communication
- Service integrations and adapters
- Voice command processing logic
- Authentication and security
